# Format JSON on Save

> VS Code extension that auto-formats JSON files on save.

## ✨ Features

- 🎨 Automatic JSON formatting on file save
- ⚙️ Zero configuration

## 🚀 Development

```bash
pnpm install
# Open in VS Code and press F5 to launch the Extension Development Host
```

## 📦 Installation

Not published to the VS Code Marketplace. Clone and install locally.
